# Testing the BDD framework and Claude skill

 small worked example of using this framework and skill for developing code by
 agenticly coding a function that calculates GC percentage. 
 
Changes in this repo go through a brief → execute → verify → commit loop rather than being made ad hoc — see `CLAUDE.md`.

## Where to look

| I want to...                                                     | Open                   |
|--------------------------------------------------------------------|-------------------------|
| Start a new session                                                 | `briefs/TEMPLATE.md` — copy it to `briefs/<NNN>-<slug>.md` |
| See what the project is supposed to do (features, scope, stack)     | `PROJECT.md`           |
| See what's already been done, and why                               | `brief-log.md`         |
| Understand the rules Claude Code follows in this repo                | `CLAUDE.md`            |
